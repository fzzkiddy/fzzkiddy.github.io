SQL 查询语句

  • 查询语句

  1. SELECT 需要查询的字段 FROM ${table} WHERE 查询条件
    * 表示查询全部字段
    ORDER BY 按指定列默认升序排列。 DESC 降序排列
    ORDER BY column_name DESC

  • SQL 函数
    SUM() 返回指定列的总数
    * SELECT SUM(column_name) FROM table_name;
    AVG() 返回指定列的平均值
    * SELECT AVG(column_name) FROM table_name;
    TOP 50 PERCENT * 返回选中百分之前50 的记录
    COUNT() 函数返回匹配指定条件的行数 
    * COUNT(*)
* COUNT(DISTINCT column_name) 返回指定列不同值的数目
    FIRST(column_name) 返回指定列记录的第一个值（只有 MS Access 支持 FIRST() 函数）
    * SQL Server、MySQL 和 Oracle 中的 SQL FIRST() 工作区
    * SELECT TOP 1 column_name FROM table_name ORDER BY column_name ASC;
    * SELECT column_name FROM table_name ORDER BY column_name ASC LIMIT 1;
    * SELECT column_name FROM table_name ORDER BY column_name ASC WHERE ROWNUM <=1;
    LAST() 返回指定列记录的最后一个值 （只有 MS Access 支持 LAST() 函数）
    * SELECT LAST(column_name) FROM table_name;
    * SELECT TOP 1 column_name FROM table_name ORDER BY column_name DESC;
    * SELECT column_name FROM table_name ORDER BY column_name DESC LIMIT 1;
    * SELECT column_name FROM table_name ORDER BY column_name DESC WHERE ROWNUM <=1;
    MAX() 返回指定列的最大值
    * SELECT MAX(column_name) FROM table_name;
    MIN() 返回指定列的最小值
    * SELECT MIN(column_name) FROM table_name;
    GROUP BY 根据一个或多个列对结果集进行分组
    * SELECT column_name FROM table_name GROUP BY column_name;
    DISTINCT 返回所选列的不同值 （去重）
    * SELECT DISTINCT column_name FROM table_name
    HAVING 函数目前还没有理解。。。！
    UNIX_TIMESTAMP
    * MySQL中的UNIX_TIMESTAMP函数有两种类型供调用
    1. 无参数调用：UNIX_TIMESTAMP() 
        返回值：自'1970-01-01 00:00:00'的到当前时间的秒数差
      例子：SELECT UNIX_TIMESTAMP()  => 1339123415
  2. 有参数调用：UNIX_TIMESTAMP(date)
      其中date可以是一个DATE字符串，一个DATETIME字符串，一个TIMESTAMP或者一个当地时间的YYMMDD或YYYMMDD格式的数字
      返回值：自'1970-01-01 00:00:00'与指定时间的秒数差
  3. 以下几种格式返回的结果相同：
      SELECT UNIX_TIMESTAMP('20120608');
      SELECT UNIX_TIMESTAMP('2012-6-8');
      SELECT UNIX_TIMESTAMP('2012-06-08');
    CURRENT_DATE() 获取当天日期 返回 YYYY-MM-DD 字符串
    NOW() 获取当前日期和时间。返回 YYYY-MM-DD hh:mm:ss

redis 基础学习 从安装到开发

Share
  • 安装
    em… 我也不记得我是怎么安装的了，这部分暂且不写
    就从系统已经有了redis 环境开始写起吧…对于如何安装redis 请翻阅他人博客。。
  • 启动

    redis-server
    当出现下图时，说明服务已启动
    1668:M 23 Jan 10:56:01.621 # Server initialized
    1668:M 23 Jan 10:56:01.622 DB loaded from disk: 0.001 seconds
    1668:M 23 Jan 10:56:01.622     Ready to accept connections

  • 启动本地服务
    默认 6379 端口 通过 –post 可以自定义端口

    redis-server –port 6380

* 通过脚本启动

  • 发送命令
    Redis 通过PING 命令来确定客户端与Redis 是否连接正常，如果连接正常会收到PONG
  • 命令返回值
    • 状态回复
      1. ok
    • 错误回复
      (ERROR)
    • 整数回复
      (integer) 0
    • 字符串回复
      “1”
      请求的键值不存在时返回一个空结果
      （nil）
    • 多行字符串返回

  • 配置

    1. 在不重新启动 Redis 的情况下，动态修改部分Redis配置
    2. 获得当前的配置活动情况

      CONFIG GET loglevel
      1）”loglevel”
      2) “warning”
      第一行字符串回复的表示选项名 、第二行字符串回复表示选项值
      SELECT table 切换数据库
      Redis不支持自定义数据库名，每个数据库都以编号命名，并且不支持分别设置密码，要么能 全部访问，要么一个数据库也访问不了

  • 创建 键 SET key 1

  • KEYS * 获取所有的键
