JSX语句

Share

JSX 是 类XML 语法的ECMAScript

JSX的基本语法

  1. 定义标签时，只允许被一个标签包裹
  2. 标签一定要闭合

    元素类型

    DOM元素和组件元素
    对应规则为 首字母大小写
  3. 大写是组件元素
  4. 小写是DOM元素
  • DOCTYPE 是一个非常特殊的标志。一般会在使用React作为服务端渲染时用到。
    在HTML中，DOCTYPE 是没有闭合的，也就是说我们无法渲染它

元素的属性

  • class 和 for 在 javascript 中都是关键词，所以 我们元素的class 和 for 属性要转换为 className 和 htmlFor
  • 组件的属性 完全自定义，可以理解为实现组件的自定义参数
    在写自定义属性的时候，都由标准写法改为小驼峰写法
  • 展开属性。如果你不知道要设置哪些props 。那么最好先不要设置 它。

    render 方法，是React 组件生命周期的一部分，也是最核心的函数之一

    HTML 转义

  • React会奖所有要显示到DOM的字符串转义，防止XSS
    1. 直接使用utf-8 字符 ©
    2. 使用对应字符的Unicode编码查询编码
    3. 使用数组组装 <div>{['cc ',<span>&copy;</span>,' 2018']}</div>
    4. 直接插入原始的HTML
      1. 此外React 提供了dangerouslySetInnerHTML属性
        它的作用就是避免React转义字符，在确定必要的情况下可以使用它
