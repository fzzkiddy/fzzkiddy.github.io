node版本控制

Share

node 版本安装

  • 使用npm 安装一个模块 n 到全局

    npm install -g n

  • 使用 n 加版本号 就可以安装其他版本了

    sudo n 6.1.0
    mac 需要sudo 权限

node 多版本切换

  • 查看node版本

    n
    (前面有个o)
    你可以通过移动上下方向键来选择要使用的版本，最后按回车生效。
    n

    0.10.1
    0.10.15
    o
    0.10.1
    0.11.8

  • 安装最新版本

    n latest

  • 安装稳定版本

    n stable

  • 删除某个版本

    n rm 0.10.1

  • 以指定的版本来执行版本

    n use 0.10.21 some.js
